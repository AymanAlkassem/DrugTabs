# DrugTabs — Swipe through your pharmacology, one tab at a time.

Flip between multiple content tabs right inside your Anki cards. No more cramming everything onto one side — organize any drug into clean, focused tabs like Indication, Contraindication, Side Effects, and Mechanism of Action. Each tab holds its own content, its own cloze, and its own space to think.

Study smarter. One tab at a time.
________________________________________________

📖 How to use:

1. Download the add-on, install via Tools → Add-ons → Install from file, and restart Anki
2. Click Add to create a new card
3. At the top, click the note type dropdown
4. Select DrugTabs from the list
5. Enter the name of the drug in the Title field, then fill in each tab with your content — for example Indication, Contraindication, Side Effects, and Mechanism of Action
6. Use cloze deletions freely within each tab, but make sure each cloze number is unique across all tabs — don't reuse the same number in different tabs
7. Any tab left empty will automatically be hidden during studying — no clutter, just the content you need
